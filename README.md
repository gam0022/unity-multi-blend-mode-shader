# unity-multi-blend-mode-shader

描画モード（ブレンドモード、カリングモード、ステンシル操作）の切り替えに対応したUnityのカスタムシェーダーのサンプルです。

## 環境

- 2018.4.13f1 (LTS)
- ビルトインレンダリングパイプライン と LightWeightRenderPipeline の両方で動作しました
